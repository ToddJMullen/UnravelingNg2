My Latest Dives (jQuery)

This was built with jQuery from the dives.js array of dive object data which is imperitive by nature. You must explicitly tell jQuery every single step to accomplish what you intend and you must provide all the logic to support each step of the way with exception of the libraries' built in utilities and whatever reusable elements you've built yourself.

By contrast, Angular uses a declarative approach in that it will accomplish the same, but will require that you express what it is that you want as the end result: “(Give me) A list of all my dive records.”
